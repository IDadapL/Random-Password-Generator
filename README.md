## Описание программы
Random Password Generator — это графическое приложение на Python с использованием Tkinter для генерации случайных паролей. Программа позволяет настраивать длину пароля и выбирать типы используемых символов (цифры, буквы, специальные символы), а также сохраняет историю сгенерированных паролей в файл. Приложение упрощает создание надёжных паролей и избавляет от необходимости придумывать их вручную.

## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `tkinter, random, string, json, os`
